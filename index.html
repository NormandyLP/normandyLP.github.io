## How to Build a Rom ?

### 1. Download Device tree of Nokia x (Normandy) by writing this code

git clone https://github.com/NormandyLP/android_device_nokia_normandy.git device/nokia/normandy

_### 2. Download Vendor files by writing this code _

git clone https://github.com/NormandyLP/android_vendor_nokia_normandy.git vendor/nokia/normandy

### 3. Download Lulz kernel by writing this code

git clone https://github.com/NormandyLP/normandy_lulz_kernel.git kernel/nokia/normandy

### 4. Download needed packages

git clone https://github.com/NormandyLP/hardware_qcom_media.git hardware/qcom/media-caf/msm7x27a

git clone https://github.com/NormandyLP/hardware_qcom_media.git hardware/qcom/media-caf/msm7x27a

### 5. Apply Patches in the device/nokia/normandy/patches

cd device/nokia/normandy/patches sh apply.sh
